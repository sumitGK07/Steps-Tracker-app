👟 StepTracker APP – Fitness Steps Tracker

A beautiful Streamlit app to track your daily steps, distance, time, and calories burned.

Features
👣 Log steps for Walking, Jogging, or Running
📏 Auto-calculates distance in km
🔥 Estimates calories burned based on your weight
⏱️ Estimates time taken
📊 7-day trend charts (steps, calories, distance)
🏆 Daily goal progress bar with tips
📋 Full activity history log
🌍 All-time total stats



before u run this app make sure u install streamlit, pandas and plotly and if it gives warning try to type this     python -m streamlit run app.py in terminal and it automatically runs it
